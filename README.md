# 24TI076.github.io
森人-morintyu- ホームページ
個人ポートフォリオサイトです。自己紹介、読んだ本の記録、コードサンプル、日記などを公開しています。

## プロジェクト概要

このプロジェクトは、HTML/CSS/JavaScriptを使用した静的な個人ホームページです。

## プロジェクト構造

24TI076.github.io/\
├── index.html          # メインHTMLファイル\
├── style.css           # スタイルシート\
├── script.js           # JavaScript機能\
├── images/             # 画像ファイル（ロゴ、スライド画像など）\
├── books/              # 読んだ本のリスト\
│   └── list.json       # 本のリストデータ\
├── codes/              # コードサンプル\
│   └──list.json        # コードリストデータ\
└── diaries/            # 日記\
    └── list.json       # 日記リストデータ

## 主な機能

### 1. スライドショー
- 自動再生機能（3秒間隔）
- 前へ/次へボタンによる手動操作
- インジケーターによる現在位置の表示
- マウスホバー時の自動再生一時停止

### 2. ナビゲーション
- スムーズスクロールによるセクション移動
- 固定ナビゲーションバー

### 3. セクション構成
- **About**: 自己紹介とサイトについて
- **Books**: 読んだ本の記録とレビュー
- **Codes**: 実装したコードサンプル
- **Diary**: 日記・学習記録

### 4. 視覚効果
- 背景の泡アニメーション

## 📝 使い方

### コンテンツの追加

#### 本の追加（Booksセクション）
\`books/list.json\`を編集して、新しい本の情報を追加してください。

#### コードサンプルの追加（Codesセクション）
\`codes/list.json\`を編集して、新しいコードの情報を追加

#### 日記の追加（Diaryセクション）
\`diaries/list.json\`を編集して、新しい日記エントリを追加してください。

### スライドショー画像の変更
\`images/\`ディレクトリに新しい画像を追加し、\`index.html\`のスライドショーセクションを編集してください。

## カスタマイズ

### スタイルの変更
\`style.css\`を編集することで、色、フォント、レイアウトなどをカスタマイズできます。

### 機能の追加
\`script.js\`に新しい機能を追加することで、インタラクティブな要素を増やすことができます。
